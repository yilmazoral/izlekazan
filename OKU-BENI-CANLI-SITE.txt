# İzleKazan v2026.05.18-022 Canlı Site Paketi

Bu paket GitHub/Render canlı site dosyalarıdır.

## Bu sürümde öne çıkanlar

- Bildirim senaryoları uygulandı.
- Web Push altyapısı eklendi.
- Paket akordeon açılmama / buton görünmeme sorunu düzeltildi.
- Üye paneli sade akordeon mantığında düzenlendi.
- Alt üyeler soy ağacı şeklinde gösterilir.
- Bildirimler ve Alt Üyeler bölüm sırası değiştirildi.

## Push bildirim için Render ortam değişkenleri

Aşağıdaki değişkenler tanımlı değilse site çalışır; fakat telefona push bildirimi gönderilmez:

```txt
VAPID_PUBLIC_KEY=
VAPID_PRIVATE_KEY=
VAPID_SUBJECT=mailto:yilmazoral@hotmail.com
```

VAPID anahtarları üretmek için bilgisayarda proje klasöründe şu komut kullanılabilir:

```bash
npx web-push generate-vapid-keys
```

Üretilen public/private key değerlerini Render > Environment bölümüne ekleyin.

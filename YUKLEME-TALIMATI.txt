İzleKazan tüm site dosyaları

GitHub'a yüklenecek dosya ve klasörler:

- index.html
- style.css
- app.js
- site-update.js
- server.js
- package.json
- supabase-setup.sql
- README.md
- assets klasörü içindeki tüm dosyalar

GitHub'da aynı isimli dosyalar varsa değiştirin. assets klasörünü de komple yükleyin.

Render'da Start Command şu şekilde kalabilir:
npm start

Render Environment değişkenleri:
SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY
JWT_SECRET
ADMIN_EMAIL
ADMIN_PASS
PUBLIC_URL

Önemli:
SUPABASE_SERVICE_ROLE_KEY sadece Render Environment bölümünde kalmalı, frontend dosyalarına yazılmamalıdır.
